# Overview
This project contains **two interactive dashboards** and full analytics built using **Excel, Power BI, Tableau, and SQL**:  
1. **Banking Analytics** – Insights into customer accounts, transactions, and bank performance.  
2. **Credit & Debit Analytics** – Analysis of credit and debit transactions, patterns, and trends.  

The project demonstrates end-to-end financial analytics, from data preparation in Excel and SQL queries to visualization in Power BI and Tableau, perfect for portfolio showcase and 
learning purposes.

# Objectives
- Prepare, clean, and analyze financial data using **Excel & SQL**.  
- Build interactive dashboards in **Power BI** and **Tableau**.  
- Monitor credit and debit patterns and banking performance.  
- Track KPIs, trends, and anomalies for decision-making.  

# Tools Used
- **Excel** – Data preparation and analysis   
- **SQL** – Data querying and extraction  
- **Power BI D** – Dashboard creation and visualization  
- **Tableau** – Additional visualization and interactive insights  

# Key Features
- Banking & financial transaction analysis  
- Credit & debit trends and patterns  
- KPI tracking and performance monitoring  
- Interactive dashboards with filters, slicers, and visuals
